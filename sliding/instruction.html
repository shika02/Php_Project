How to use it?

  1. Download the package and unzip it
  2. Open index.html and check that everything works fine
  3. Copy to your directory keyframes.css and add it in your .html file
  4. Add id="show" to the div with slides (.slide)
  5. Put images in that .slide div, and if you need description use p.desc after the img
  6. To add more than 5 slides go to the keyframe.css and just add it
  7. To change the displaying time change number of seconds in lines like that animation: img1 40s infinite; in keyframes.css

Have fun!
